DES for Java Notes

This page is some notes about the Java implementation of the DES.

Implementation Notes

The nz.ac.auckland.cs.des package.

Julian Harris ported the C DES package from UNIX to the Macintosh, then converted it to Java.

How does the DES package work?

Very well thank you. Seriously, you'll have to ask Julian for all of the details.

Many of the class files consist of initialised static final (constant) arrays and represent the key schedule of the DES scheme.

C_BlockConverts data to and from 8-byte blocks.
des_encryptEncrypts the data blocks.
des_hash_initInitialises the encryter.
Key_scheduleHandles the key schedule.
uHexConverts data to and from hexadecimal.
desArrayInputStreamInputs a stream of encrypted data from an array.
betterArrayOutputStreamOutputs a stream to an array.
desDataInputStreamInputs a stream of encrypted data.
desDataOutputStreamOutputs a stream of encrypted data.

How do I use the DES package?

You shouldn't (?) need to use the DES package directly - it's used by the packet processor.
